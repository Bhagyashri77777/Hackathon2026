TrustShield: Multi-Modal Fraud Detection System
Team: CodeShield Alliance

Project Status: Phase 1 - Prototype Design (Round 1 Submission)

 Overview
TrustShield is an AI-powered safety engine designed to detect phishing URLs and Deepfake audio in real-time. Our goal is to provide "Zero-Effort" digital safety for senior citizens and first-time internet users.

 Tech Stack (Round 2 Roadmap)
Frontend: HTML5, CSS3, JavaScript

Backend: Python (Flask/FastAPI)

ML Models: Scikit-Learn, Pre-trained Audio Analysis APIs

Developed for the College Hackathon 2026. Codebase deployment beginning in Round 2.
